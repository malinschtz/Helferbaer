## 01: Helfer Evaluation

### Meta

Status
: **Work in progress** - Done - Obsolete

Updated
: 01.02.2026

### Goal

Wie schnell können sich Helfer registrieren und verstehen sie intuitiv, wie sie danach vorgehen müssen, um ihren ersten Job zu finden und zu buchen?

### Methode

+ Testpersonen unter 60 ohne Vorkenntnisse über die Applikation 
+ Screen-Recording um Zeit und Clicks später auszumessen + Think-Aloud-Protokoll für mehr Einblicke in die Gedankenabläufe
+ Aufgaben: Registrieren -> Dashboard -> Job Suchen -> Job buchen
+ Zusätzlich Fragen: Dashboard/Stundenkonto verständlich?
+ Bewertung: 
    - Zeitmessung pro Task 
    - Anzahl Klicks pro Task
    - Formularfehler und Validierungsfehlermeldungen zählen
    - Thik-Aloud-Protokoll auswerten
