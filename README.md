# Helferbaer
Web Applikation, die eine Plattform als Schnittstelle zwischen Alttagshelfer und Menschen ab Pflegegrad 1 bietet.

# Schritte um die App zu starten:

1. Erstelle und aktiviere eine [Python Virtual Environment](https://realpython.com/python-virtual-environments-a-primer/#how-can-you-work-with-a-python-virtual-environment)

2. Installiere alle benötigten Packages mithilfe des Terminal und dem Command `pip install -r requirements.txt`

3. Initialisiere die Datenbank der App mit dem Command `flask init-db` und füge die Sample-Daten mit dem Command `flask insert-sample` ein

4. Starte den Web-Server mit dem Command `flask run`
